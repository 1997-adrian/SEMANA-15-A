📋 Lista de Tareas - Aplicación de Escritorio
Una aplicación de escritorio moderna para gestionar tareas diarias, desarrollada con Python y Tkinter, siguiendo arquitectura modular por capas.
✨ Características
🔐 Sistema de Login con autenticación de usuarios
📝 Gestión de Tareas (crear, completar, eliminar)
💾 Persistencia de Datos en formato JSON
🎨 Interfaz Gráfica Moderna con Tkinter
⌨️ Eventos de Teclado y Ratón (Enter, Doble Clic)
📦 Empaquetado como Ejecutable con PyInstaller
🏗️ Arquitectura Modular por capas (UI, Servicios, Modelos)
